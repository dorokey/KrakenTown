# KrakenTown

In the second commit, the game now has most of its basic functions. You collect PowerUps that help you, and keys to reach the next Level, you get a score and a message each for a game either lost or won. Most of the thing tryin to harm the player change each level (except for the jellyfish because I thought they were really cool looking).
There is an enemy at the top that needs to be defeated to win the game so that it doesnt go on endlessly, but one can still try to beat the score.
